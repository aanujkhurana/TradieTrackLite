# Legacy Backend

This Express/MongoDB backend is retained as optional legacy code from the earlier backend-first direction.

The local-first MVP does not require this backend for normal app use. Jobs, photos, reminders, reports, ads, and the ad-free purchase boundary are handled in the Expo app.

Use this folder only when maintaining old backend tests, comparing legacy behavior, or intentionally experimenting with future optional backend features. Do not make it a requirement for mobile job workflows.

```sh
npm install
npm test
```
